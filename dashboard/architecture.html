데이터 항목

항목 XML 태그 수집 방식 활용
기자명 <writer> 자동 기자별 기사 집계
면(Page) <pageNumber> 자동 1면/주요면 가중치
글자수 <content> 길이 자동 기사 분량 측정
원문 링크 <link> 자동 기사 원문 확인
제목 <title> 자동 기사 식별
발행일 <pubDate> 자동 기간별 분석
옵션1 (톱/단독) - 수동 특종/주요기사 가산점
옵션2 (부장평가) - 수동 정성평가 점수
옵션3 (기타) - 수동 추가 메모/가산점

IT 지원부 요청사항

Firebase Storage 접근 권한
  • Firebase 프로젝트 접근 권한 부여
  • Storage daily/ 폴더 읽기 권한
  • 서비스 계정 키(JSON) 발급
XML 데이터 구조 문서
  • XML 스키마 또는 샘플 파일
  • 각 태그별 설명 문서
  • 데이터 업데이트 주기 확인
참고: 현재 Firebase Storage의 daily/ 폴더에 XML 데이터가 매일 자동 업로드되고 있음을 확인했습니다. 해당 데이터에 접근할 수 있는 권한만 있으면 시스템 구축이 가능합니다.
